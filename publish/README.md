# Publishing

This folder is the **ready-to-post** catalog.

- `dossiers/` holds research, sources, timelines, and working briefs.
- `publish/` holds polished drafts formatted for posting (Substack, site, etc.).

## Substack

Queue: `publish/substack/queue/`

Conventions:
- Filenames: `YYYY-MM-DD-slug.lang.md` (e.g. `2026-03-15-control-surfaces.lang=en.md`)
- Each file should be copy/paste-ready for Substack.
- Keep a `source:` pointer to the originating dossier/brief when applicable.
